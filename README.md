# 🚀 ShopIQ – AI Powered E-Commerce Platform

ShopIQ is a modern full-stack AI-powered e-commerce web application built using React, Tailwind CSS, Django REST Framework, and JWT Authentication.

---

## ✨ Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 🛍️ Dynamic Product Listing
- 📦 Product Details Page
- 🛒 Shopping Cart System
- ❤️ Add / Remove Cart Items
- 🔍 Real-Time Product Search
- 🤖 AI-Style Product Recommendations
- 🔒 Protected Routes
- ⚡ Responsive Modern UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router
- Axios
- Context API

### Backend
- Django
- Django REST Framework
- JWT Authentication
- SQLite
